# PDF Layout System

## Structure

1. Title
2. Executive Summary
3. Methodology and Data Window
4. Major Findings
5. Tables and supporting evidence
6. Recommendations and Next Actions

## Section Pattern

Use this pattern inside each major section:

- Insight
- Why it matters
- Action

Keep each part short and direct. If a paragraph starts to sprawl, convert it into bullets.

## Writing Rules

- Keep paragraphs to 2-3 lines when possible.
- Use bold sparingly for emphasis only.
- Prefer bullets for comparisons, callouts, and action lists.
- Avoid duplicate explanations across sections.

## Page Break Rules

- Break after a complete section.
- Break before a major table block.
- Break before a new topic only when the previous section is complete.
- Never break between a heading and its content.
- Never break inside a table.

## Table Rules

- Keep tables simple and compact.
- Put the table right after the section that explains it.
- Keep numeric values visually clean and easy to scan.
- Avoid nested tables or excessive notes inside table cells.

## Image Rules

- Use local image paths only.
- Keep images large enough to inspect.
- Place images close to the supporting insight.
- Avoid tiny thumbnails and remote links.

## Final Check

Before export, verify the draft reads like a board-ready report:

- scannable
- insight-first
- concise
- visually organized
- free of raw export noise
